# Repository map

This repository is the companion reproducibility repository for:

**Spectral Organization of a 3D Helmholtz–Robin Field Core and Its Transition into Measurement Geometry**

It begins from the validated numerical basis established in:

**Reproducible Low-Mode Ordering in a 3D Helmholtz–Robin Eigenproblem on Cube-Like and Lattice-Linked Geometries**

Recommended reading order:

1. **00_inherited_foundation** — what is inherited and not re-proved here
2. **01_field_core_ordering** — dense beta tracking, subspace robustness, axis permutation robustness, readout robustness
3. **02_nullmodels_and_phase_sensitivity** — null-model attacks and phase-sensitive diagnostics
4. **03_carrier_opposite_cef_tests** — opposite-pair locking, carrier shells, corner-edge-face incidence, cross-layer effects
5. **04_reduction_to_minimal_core** — reduction from many diagnostics to a smaller field-core backbone
6. **05_field_core_function** — transition-zone and carrier-space interpretation
7. **06–10** — gradual build-up from slit baselines to the effective detector law and microscopic Robin-noise support

The repository should therefore be read as a bridge from validated low-mode ordering to measurement geometry, not as an independent starting point.
