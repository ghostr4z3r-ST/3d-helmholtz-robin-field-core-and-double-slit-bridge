Double-slit with explicit local Robin modification at slit walls

Model
- 2D slit-channel Helmholtz surrogate
- each slit treated as a short channel of width a and thickness L
- slit-wall boundary condition: ∂n u + β u = 0 on the slit walls
- channel output uses the fundamental Robin transverse mode q tan(q a/2)=β
- coherent propagation from slit exit to screen via Fresnel kernel

Summary
- Deterministic real Robin changes mostly preserve fringes until the modified slit approaches cutoff; then the pattern collapses mainly because that slit is nearly blocked.
- Stochastic local Robin fluctuations at one slit suppress fringe visibility continuously and move the screen profile toward the incoherent slit-sum.

Interpretation
- A purely deterministic local Robin change is not enough to generically erase interference; it mainly phase-shifts or attenuates one slit.
- A local Robin coupling that fluctuates microscopically / remains unresolved at one slit produces effective decoherence and drives the profile toward the incoherent slit-sum.
