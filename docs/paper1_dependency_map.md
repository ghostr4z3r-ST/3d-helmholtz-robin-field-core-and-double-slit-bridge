# Paper 1 dependency map for Repository 2

Repository 2 begins from the validated basis established in **Paper 1 / Repository 1**.

## What Repository 2 inherits
The following are treated as already established by Paper 1:
- a 3D Helmholtz–Robin field model on cube-like and lattice-linked geometries,
- artifact-hardened ordering in that model,
- the basic Helmholtz / Robin role separation,
- the vocabulary of field-core, readout, and lattice-linked ordering,
- and the general null-model discipline used to test whether observed structure is numerically real.

## What Repository 2 adds
Repository 2 does not repeat the Paper-1 task. Instead it adds:
- sharper field-core organization under Robin variation,
- family-phase hardening,
- opposite / carrier / CEF bridge diagnostics,
- reduction toward a smaller field-core description,
- and the stepwise continuation into the double-slit / detector law.

## Practical reading rule
If a reader wants to understand **why robust ordering is taken seriously at all**, they should start with Paper 1 / Repository 1.
If a reader accepts the Paper-1 basis and wants to understand **how that ordering is organized and extended into measurement geometry**, they should read Repository 2.

## Suggested citation bridge
When Repository 2 is made public, its README and eventual Paper-2 manuscript should explicitly cite:
- the Paper-1 manuscript,
- the Paper-1 GitHub repository,
- and the Paper-1 Zenodo record.
