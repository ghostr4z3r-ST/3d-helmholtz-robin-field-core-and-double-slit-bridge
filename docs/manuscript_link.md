# Manuscript link

This repository is the companion reproducibility repository for the manuscript:

**Spectral Organization of a 3D Helmholtz–Robin Field Core and Its Transition into Measurement Geometry**

At the present stage the manuscript is maintained as a preprint draft. Once the manuscript receives its public preprint identifier, this file should be updated with the final arXiv and/or DOI citation.

It builds explicitly on the earlier companion study:

**Reproducible Low-Mode Ordering in a 3D Helmholtz–Robin Eigenproblem on Cube-Like and Lattice-Linked Geometries**

Companion repository / archive of the earlier study:
- GitHub: `ghostr4z3r-ST/3d-helmholtz-robin-ordering`
- Zenodo DOI: **10.5281/zenodo.19335898**

The earlier study establishes the numerical existence and artifact-hardening of low-mode ordering. The present repository starts from that validated basis and reproduces the spectral-organization, bridge, double-slit, detector-law, and modified-setup analyses used in the current manuscript.
