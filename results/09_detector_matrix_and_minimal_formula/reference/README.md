# Detector matrix and minimal formula

This block contains the effective detector-law data products used to reduce the detector response to a smaller empirical law of the form described in "Spectral Organization of a 3D Helmholtz–Robin Field Core and Its Transition into Measurement Geometry".

This repository version already contains the detector-law bridge structure, but some file groups may remain historically organized across reports and figures rather than as a single polished release pack.
