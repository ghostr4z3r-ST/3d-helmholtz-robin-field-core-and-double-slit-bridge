# Inherited foundation

This module records the minimal inherited basis from the earlier companion manuscript:

**Reproducible Low-Mode Ordering in a 3D Helmholtz–Robin Eigenproblem on Cube-Like and Lattice-Linked Geometries**

The analyses in this repository do not re-prove from scratch that low-mode ordering exists in the 3D Helmholtz–Robin model. They assume that validated basis and continue from it toward spectral organization and measurement geometry.
